# Current weather condition
The application displays information about weather in the choice city with the location on the map.

### Tech used
* HTML
* CSS
* JavaScript
* Leaflet

### Sources 
* Map's layer:
https://openstreetmap.org/
* Weather condition:
https://openweathermap.org/
